# DRV8323 — Gate Driver SPI Klasse

Minimale SPI-Helper-Klasse für den **DRV8323S Gate Driver** (Texas Instruments).

## Hardware

- IC: DRV8323SRTAR
- Interface: SPI (bis 1 MHz)
- Kontrolliert 3-Phasen-Halbbrücke für BLDC

## SPI-Pins (via [[rc_pins]])

| Signal | GPIO |
|---|---|
| MISO | 5 |
| MOSI | 6 |
| SCLK | 7 |
| CS | 15 |
| EN | 16 |
| nFAULT | 39 |

## API

| Methode | Beschreibung |
|---|---|
| `begin(spiHz)` | Pins init, SPI start, Driver aufwecken |
| `readRegister(reg)` | 11-bit Register lesen |
| `writeRegister(reg, val)` | 11-bit Register schreiben |
| `hasFault()` | nFAULT-Pin abfragen |
| `readFault1()` | Fault Register 1 lesen |
| `readFault2()` | Fault Register 2 lesen |
| `clearFaults()` | EN Puls → Latched Faults löschen |

## Verwendung

```cpp
DRV8323 drv(PIN_DRV_CS, PIN_DRV_EN, PIN_DRV_FAULT,
            PIN_DRV_SCLK, PIN_DRV_MISO, PIN_DRV_MOSI);

drv.begin(1000000);  // 1 MHz SPI

if (drv.hasFault()) {
    uint16_t f1 = drv.readFault1();
    drv.clearFaults();
}
```

→ [[rc_motor]] verwendet diese Klasse
